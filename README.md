# Sketch2Sim

This repository is for the dataset and implementation associated with the manuscript:

**“Sketch2Sim: An End-to-End Modular Framework for Hand-drawn Circuit Analysis.”**

## Status

This repository has been created to support transparency and reproducibility of the study.  
The dataset, annotations, and implementation will be released here upon publication of the paper.

## Planned Contents

The repository will include the following materials:

- **Dataset of hand-drawn circuit diagrams**
- **Component detection annotations**
- **Text-region detection annotations**
- **OCR labels for handwritten component values**
- **Implementation of the end-to-end pipeline**
- **Instructions for reproducing the main experiments**

## Dataset Summary

The study is based on a custom dataset of 1000 hand-drawn circuit diagrams focused on first-order RLC circuits with independent sources.  
Additional task-specific annotations were prepared for:
- component detection,
- text-region localization, and
- OCR-based value recognition.
